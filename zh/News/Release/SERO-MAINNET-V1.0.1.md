* **RC3**
  * [x] 修改了 Seed2Sk 的 JSONRPC 接口的兼容性 Bug
  * [x] 修改了可能导致链分叉的一个潜在Bug
* **RC2**
  * [x] 修改了一些兼容性的Bug
* **RC1**
  * [x] SuperZK2.0 加密协议
  * [x] 去中心化轻钱包交互接口
  * [x] 去除POW许可证
  * [x] 优化数据同步和生成交易证明的性能
